# 🎨 Cista Sketcher


![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Turtle Graphics](https://img.shields.io/badge/Turtle-Graphics-green?style=for-the-badge)


An interactive drawing application built with Python Turtle Graphics that allows users to create sketches using keyboard controls.

---

## 📖 About

Cista Sketcher is a simple drawing application created using Python's Turtle Graphics library. Users can customize the drawing cursor, choose colors, change the background, and create sketches by moving the pen across the screen using keyboard controls.

This project is ideal for beginners learning Python graphics, event handling, and GUI programming.

---

## ✨ Features

- 🎨 Multiple drawing colors
- 🖌️ Custom turtle shapes
- 🌈 Customizable background colors
- ⌨️ Keyboard-controlled drawing
- 🔄 Direction and rotation controls
- ❓ Built-in help window
- 🐢 Powered by Python Turtle Graphics

---

## 🚀 Getting Started

### Requirements

- Python 3.x

The following modules are used:

- `turtle`
- `tkinter`

Both come pre-installed with standard Python installations.

---

## ▶️ Run the Application

Clone the repository:

```bash
git clone https://github.com/your-username/cista-sketcher.git
```

Move into the project folder:

```bash
cd cista-sketcher
```

Run the application:

```bash
python cista_sketcher.py
```

## 🎮 Controls

### Movement

| Key | Action |
|------|---------|
| `Y` | Move Up |
| `B` | Move Down |
| `G` | Move Left |
| `H` | Move Right |

### Rotation

| Key | Action |
|------|---------|
| `U` | Rotate 45° Left |
| `N` | Rotate 45° Right |
| `T` | Rotate 135° Left |
| `V` | Rotate 135° Right |

### Forward Drawing

| Key | Action |
|------|---------|
| `I` | Move Forward |
| `M` | Move Forward |
| `R` | Move Forward |
| `C` | Move Forward |

### Help Window

- Click anywhere on the screen to open the Help Menu.
